# 🚀 Restfull Api Outven 게임 커뮤니티 게시판 프로젝트

## 📌 프로젝트 개요  
게임 커뮤니티 inven을 모티브로 하여 Restfull Api 서비스를 만듦

## 🛠 기술 스택
    O/S​ : Windows 10 Home (64bit)​
    Language​ : Java, HTML, CSS, ​
    script​ : Javascript, jQuery, Ajax​
    Server​ : Tomcat 11.​
    Database​ : Oracle database 18 express​
    Framework​ : Spring Tool Suite4​
    Library​ : Lombok 1.18.30, ojdbc8 23.3.0.23.09,spring Data JPA, spring WEB, spring mustache​



## 📌 STS에 JPA 적용
    1. Help 메뉴  ⇒ Install new software
    2. Work with: 항목에 http://download.eclipse.org/releases/oxygen를 입력 후 엔터
    3. 아래 Name 항목 여럿 중에서 "Web, XML, Java EE and OSGi Enterprise Development" 항목을 펼쳐서 JPA 관련 모든 항목을 체크한다
      (이때 간단히 찾는 방법은 위의 "type filter text" 항목에 JPA라고 입력)
    4. 이렇게 설치과정을 진행 후 이클립스 재구동
